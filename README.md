# Zane-Maverick

这里整理的是我关注、研究和可能自部署的一组开源工具，主要集中在 AI 网关、模型 API 聚合、个人 Agent、自动化工具和日常实用软件。

## AI 网关与 API 聚合

- [new-api](https://github.com/Zane-Maverick/new-api) - 统一 AI model hub，支持多模型服务转 OpenAI / Claude / Gemini 兼容接口。
- [sub2api](https://github.com/Zane-Maverick/sub2api) - 订阅统一接入与中转服务，适合把多类 AI 订阅整理成统一入口。
- [one-api](https://github.com/Zane-Maverick/one-api) - LLM API 管理与分发系统，适合统一管理 key、渠道和二次分发。
- [GPTSession2CPAandSub2API](https://github.com/Zane-Maverick/GPTSession2CPAandSub2API) - GPT session 到 CPA / Sub2API 工作流相关工具。
- [ChatGPTRegister](https://github.com/Zane-Maverick/ChatGPTRegister) - ChatGPT 注册与自动化相关工具。

## Agent 与个人助手

- [openclaw](https://github.com/Zane-Maverick/openclaw) - 跨平台个人 AI assistant。
- [chatgpt-on-wechat](https://github.com/Zane-Maverick/chatgpt-on-wechat) - CowAgent / 微信与企业协作平台接入方向的 AI 助手。
- [awesome-openclaw-usecases](https://github.com/Zane-Maverick/awesome-openclaw-usecases) - OpenClaw 使用场景集合。

## 开发、前端与数据工具

- [react-bits](https://github.com/Zane-Maverick/react-bits) - 可定制的 React 动效组件集合。
- [Scrapling](https://github.com/Zane-Maverick/Scrapling) - 自适应 Web scraping / crawling 框架。
- [tduck-survey-form](https://github.com/Zane-Maverick/tduck-survey-form) - 自托管表单与问卷系统。

## 游戏、网络与系统工具

- [Playnite](https://github.com/Zane-Maverick/Playnite) - 游戏库统一管理器。
- [ludusavi](https://github.com/Zane-Maverick/ludusavi) - PC 游戏存档备份工具。
- [clash-verge-rev](https://github.com/Zane-Maverick/clash-verge-rev) - 基于 Tauri 的代理 GUI 客户端。
- [VPN](https://github.com/Zane-Maverick/VPN) - SSPanel-UIM 自托管代理服务管理面板 fork。

## 当前整理原则

- 优先保留与 AI 工具栈、自部署、自动化和实用软件相关的仓库。
- Fork 仓库保持和上游同步，避免长期落后。
- 用 topics 做检索标签，用 Profile README 做人工分类索引。
- 对仓库名不准确的项目，后续再统一改名，避免影响已有链接习惯。
